# &	: Bitwise AND :	x & y : Returns 1 if both the bits are 1 else 0
# |	: Bitwise OR : x | y : Returns 1 if either of the bit is 1 else 0
# ~	: Bitwise NOT : ~x : Returns one’s compliement of the number, ~x - Python Ones’ complement of a number ‘x’ is equals to -(x+1).
# ^	: Bitwise XOR : x ^ y : Returns 1 if one of the bits is 0 and the other bit is 1. If both the bits are 0 or 1, then it returns 0.
# >> : Bitwise right shift : x>> : Shifts the bits of the number to the left and fills 0 on voids left as a result. Similar effect as of multiplying the number with some power of two.
# << : Bitwise left shift : x<< : Shifts the bits of the number to the right and fills 0 on voids left as a result. Similar effect as of dividing the number with some power of two.
# ------------------------------------------------------------------------------
# a = 10 = 1010 (Binary)
# b = 4 =  0100 (Binary
# a & b = 1010 & 0100 = 0000 = 0 (Decimal)
# a | b = 1010 | 0100 = 1110 = 14 (Decimal)
# ~a = ~1010 = -(1010 + 1) = -(1011) = -11 (Decimal)
# a ^ b = 1010 ^ 0100 = 1110 = 14 (Decimal)
# a >> 2 = 1010 >> 2 = 0010 = 2 : Right Shift : Add number(given at right side of sign >> ) of zeros to left side and cancel the right numbers of same lenght.
# a << 2 = 1010 << 2 = 101000 = 40 : Left Shift : Add number(given at right side of sign >> ) of zeros to Right side.
